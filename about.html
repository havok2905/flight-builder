Why did I make this?

I made this app to explore a few various things I've been studying in my free time.

  • React
  • Algorithms
  • Data Structures

I wanted to see how I can bridge these three topics in a way that was interesting and fun to work on. I'm a huge craft beer nut, so this was a natural fit!

Goals

  • Write all application code in ES6+
  • Approach the app from a service oriented perspective
  • Keep view logic thin and concerned only with maintaining component state
  • Make use of one traditional data structure
  • Make use of one comparison algorithm in the form of a custom NPM module
  • Encapsulate one graph type in a React component
